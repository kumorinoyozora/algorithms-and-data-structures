1. Загрузка графа

 Загрузка существующего файла с корректным графом. +

 Отмена загрузки → генерация графа. +


2. Генерация графа

 Ввод валидных значений (напр., 5 вершин, плотность 0.5). +

 Ввод 0 вершин. +

 Ввод 1 вершины. +

 Плотность = 0, 100, больше 100, отрицательная — проверка валидации. +

 Проверка генерации графа с минимальной и максимальной плотностью рёбер. +


3. Работа основного меню

 Добавление вершины. +

 Добавление ребра (между существующими, к несуществующей). +

 Добавление и удаление петли. +

 Удаление вершины. +

 Удаление ребра (существующего / несуществующего). +

 Выход из программы. +


4. Поиск

DFS

 Поиск из существующей вершины. +

 Поиск из несуществующей. +

 Поведение при цикле в графе. +

 Проверка подсветки пути на .png. +

Bellman-Ford

 Путь от A до B (существует). +

 Путь от A до B (не существует). +

 Проверка .png. +

Floyd-Warshall (3 кратчайших пути)

 3 или больше кратчайших путей. +

 1 или 2 пути — должен быть понятный вывод (или проверка на count). +

 0 путей — граф несвязный. +


5. Визуализация

 Файл .dot создаётся корректно. +

 Файл .png генерируется и отображает нужные цвета и рёбра. +

 Цвета трёх путей не перекрываются, а добавляются последовательно. +


6. Пограничные случаи

 Пустой граф (0 вершин). +

 Граф с одной вершиной. +

 Полносвязный граф. +
